## 概述

本项目为html+css实现单标签天气样式

## 使用

### 安装

```shell
# 克隆项目
git clone git@github.com:mitudegaoyang/weather.git
```

## 开发

### 目录结构

```text
weather
├── assets                  # 静态资源目录
│   ├── css                 # 样式文件
│   │   ├── index           # 公共样式
│   │   ├── Sunny           # 晴天样式
│   │   ├── Cloudy          # 多云样式
│   │   ├── Rainy           # 下雨样式
│   │   └── Snowy           # 下雪样式
│   └── index.js            # 公共
├── index.html              # 首页入口文件
└── README.md               # 项目说明文档(markdown格式)
```

### 主要实现功能

* 晴天
* 多云
* 下雨
* 下雪
